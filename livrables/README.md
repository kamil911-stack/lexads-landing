# Livrables

Ce dossier contient tous les outputs produits par Claude ou par toi dans le cadre de l'agence SEA.

**Règle d'or :**
- Les **inputs** (documents que tu fournis : briefs, exports, PDFs) vont dans `context/import/`
- Les **outputs** (ce que Claude produit pour toi) vont ici dans `livrables/`

---

## Organisation

| Dossier | Contenu |
|---------|---------|
| `clients/` | Livrables spécifiques à chaque client : audits, rapports de performance, comptes rendus |
| `prospection/` | Séquences email, scripts d'appel, messages LinkedIn, listes de prospects |
| `offres/` | Offres de service, grilles tarifaires, propositions commerciales |
| `campagnes/` | Structures de campagnes Google Ads, listes de mots-clés, templates d'annonces |
| `outils/` | Scripts, automatisations, templates de reporting réutilisables |
| `communication/` | Posts LinkedIn, contenus de preuve sociale, calendrier éditorial |
| `site-web/` | Textes des pages, structure, SEO, briefs visuels, code |

---

## Convention de nommage

```
AAAA-MM-JJ_nom-client_type-livrable.ext
```

Exemples :
- `2026-05-22_cabinet-dupont_audit-seo.md`
- `2026-06-01_prospect-seq-avocats-v1.md`
- `2026-05-22_offre-starter-avocats.md`
