# campagnes/

Ressources techniques pour les campagnes Google Ads.

Contenu type :
- Structures de campagnes (Search, Display, LSA)
- Listes de mots-clés par niche (droit civil, pénal, travail salarié)
- Templates d'annonces (titres, descriptions, extensions)
- Stratégies d'enchères
- Checklists de lancement et d'optimisation

Ces ressources sont réutilisables d'un client à l'autre.
